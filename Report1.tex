\section* {Постановка задачи}
\parindent=1cm

\hspace{\parindent} Требуется разработать программный пакет для редактирования шрифтов и просмотра результатов с возможностью тестовой печати текста созданным шрифтом. Пользователь должен иметь возможность:
\begin{itemize}
  \item Создать новый шрифт
  \item Редактировать внешний вид букв шрифта (буквы задаются набором кривых Безье) 
  \item Сохранить внесённые изменения с возможностью последующей загрузки
  \item Опробовать шрифт во встроенном текстовом редакторе. 
\end{itemize}

\section* {Используемая модель}
\hspace{\parindent}Для решения поставленной задачи была применена теория кривых Безье. Напомним, что кривая Безье m-го порядка задаётся параметрическим уравнением
$$R(t)=\sum\limits_{i=0}^{m}{m \choose i} t^i(1-t)^{m-i}P_i, t \in [0,1]$$
где $P_i \in \mathbb{R}$ - базисные точки кривой.
Важным при использовании кривых является понятие геометрической непрерывности. \\
\textbf{def } Кривая называется $G^0$-непрерывной, если она непрерывна. \\
\textbf{def } Кривая называется $G^1$-непрерывной, если она непрерывна, если её касательная меняется непрерывно при изменении параметра. \\
\textbf{def } Кривая называется $G^2$-непрерывной, если она $G^1$-непрерывна и её вектор кривизны меняется непрерывно. \\

Заметим, что при разработке шрфитов наиболее интересны геометрически непрерывные кривые разных порядков, ибо они позволяют достичь необходимоых виузальный свойств кривой. В частности, границы букв должны состоять из $G^0$-кривых. Если требуется гладкость границ, то стоит использовать $G^1$-кривые. Очевидно также, что на плоскости любая \\$G^1$- кривая будет $G^2$-кривой

В нашей работе мы использовали кривые Безье третьего порядка, задаваемые четырьмя базисными точками на плоскости. Нетрудно понять, что любая кривая Безье априори является $G^1$-непрерывной. Однако, чтобы таким свойством обладала кривая, составленная из кривых Безье, требуется выполнение определённых граничных условий. 

Для $G^1$-непрерывности составной кривой требуется, чтобы в точке сочленения направляющие векторы касательных были коллинеарны, что возможно только при условии нахождения трех последовательных крайних точек (точки сочленения и её соседей) на одной прямой.  
 
 Для хранения модели были реализованы два класса. Класс \tt{Curve} \normalfont преставляет собой простую модель кривой Безье, а класс \tt{DrawableCurve}\normalfont\space- его обёртку, предоставляющую необходимые интерфейсы для отображения кривой на экране. Составная кривая хранится как объект, содержащий список объектов \tt{DrawableCurve}\normalfont\space, а также вспомогательную информацию о составной кривой (например, замкнута ли кривая).

\section* {Тестирование программы}
Был проведен ряд тестов, в ходе которых оба алгоритма выдавали верное решение с точностью $10^{-5}$ (в общем случае точность зависит от абсолютных величин коэффициентов матриц и векторов, подаваемых на вход, чем они больше, тем хуже точность из-за неравномерности машинной сетки чисел с плавающей точкой, которые используются при решении). 
Рассмотренные тесты:\\
\noindent test1:\\
answer: x = 5.5, y = 0.5\\
c = (1, 1)\\
y >= -x + 6;\\
x >= y + 5;\\
y >= -x + 6;\\
x >= y + 5;\\

\noindent test2:\\
answer: y = 1, x = 2\\
с = (1, 1)\\
y >= 4 - 1.5x1;\\
y >= 0.5x1;\\
y >= 2x1 - 6;\\

\noindent test3:\\
answer: y = 2, x in [2, 4]\\
с = (1, 1)\\
2 - y <= 0;\\
2x + y >= 6;\\
y - 2x >= -6;\\

\noindent test4:\\
answer: 0 0 5\\
c = (-2, -3, -4)\\
3x + 2y + z <= 10;\\
2x + 5y + 3z <= 15;\\
Все тесты были успешно пройдены, при этом оба метода выдавали одинаковые результаты (с точностью $10^{-5})$.
