\section* {Постановка задачи}
\parindent=1cm

\hspace{\parindent} Требуется разработать программный пакет для создания и редактирования векторных шрифтов, разработать инструмент для ввода текста созданным шрифтом с последующим сохранением и редактированием.

Техническое задание:
Редактор шрифтов должен обеспечивать выполнение следующих функций:
\begin{itemize}
  \item Создание шрифта
  \item Редактирование внешнего вида букв шрифта (буквы задаются набором кривых Безье)
  	задание контрольных точек кривых
  	изменение положения точек
  	изменение положения кривых целиком
  	изменение положения набора связанных кривых одновременно
  	подразбиение кривых
  	выбор гладкости
  	заливка пространства между кривыми
  	отмена последнего действия
  \item Сохранение внесённые изменения с возможностью последующей загрузки и редактирования
  \item Возможность пробовать шрифт во встроенном текстовом редакторе. 
\end{itemize}
Редактор текста должен обеспечивать выполнение следующих требований:
\begin{itemize}
  \item Загрузка шрифта, разработанного в редакторе шрифтов
  \item Набор текста с клавиатуры
  \item Возможность редактирования текста
  	изменение любого символа набранного текста
  	изменение размера отдельных символов текста
  	использование нескольких шрифтов в тексте
  \item 
  \item Сохранение и загрузка текста с возможностью последующего редактирования
  	разработать формат файла для эффективного хранения и загрузки текста.
\end{itemize}

\section* {Используемая теоретическая модель}
\hspace{\parindent}Для решения поставленной задачи была применена теория кривых Безье. Напомним, что кривая Безье m-го порядка задаётся параметрическим уравнением
$$R(t)=\sum\limits_{i=0}^{m}{m \choose i} t^i(1-t)^{m-i}P_i, t \in [0,1]$$
где $P_i \in \mathbb{R}$ - базисные точки кривой.
Важным при использовании кривых является понятие геометрической непрерывности. \\
\textbf{def } Кривая называется $G^0$-непрерывной, если она непрерывна. \\
\textbf{def } Кривая называется $G^1$-непрерывной, если она непрерывна, если её касательная меняется непрерывно при изменении параметра. \\
\textbf{def } Кривая называется $G^2$-непрерывной, если она $G^1$-непрерывна и её вектор кривизны меняется непрерывно. \\

Заметим, что при разработке шрифтов наиболее интересны геометрически непрерывные кривые разных порядков, ибо они позволяют достичь необходимых визуальный свойств кривой. В частности, границы букв должны состоять из $G^0$-кривых. Если требуется гладкость границ, то стоит использовать $G^1$-кривые. Очевидно также, что на плоскости любая \\$G^1$- кривая будет $G^2$-кривой

В нашей работе мы использовали кривые Безье третьего порядка, задаваемые четырьмя базисными точками на плоскости. Нетрудно понять, что любая кривая Безье априори является $G^1$-непрерывной. Однако, чтобы таким свойством обладала кривая, составленная из кривых Безье, требуется выполнение определённых граничных условий. 

Для $G^1$-непрерывности составной кривой требуется, чтобы в точке сочленения направляющие векторы касательных были коллинеарны, что возможно только при условии нахождения трех последовательных крайних точек (точки сочленения и её соседей) на одной прямой.  
 
Для хранения модели кривой Безье были реализованы два класса. Класс \tt{Curve} \normalfont представляет собой простую модель кривой Безье, а класс  Модели-Представления  \tt{DrawableCurve}\normalfont\space- его обёртку, предоставляющую необходимые интерфейсы для отображения кривой на экране. Составная кривая хранится как объект, содержащий список объектов \tt{DrawableCurve}\normalfont\space, а также вспомогательную информацию о составной кривой (например, замкнута ли кривая).

Класс\space\tt{Letter}\normalfont\space представляет модель буквы шрифта. 
Символ хранится в формате TrueType, созданном для хранения информации o контуре символа (векторное изображение). В отличие от растрового символа — векторный легко масштабируется. Но и у векторного способа хранения есть свои недостатки. Для того чтобы однозначно понимать (интерпретировать) записанную информацию, пользуются следующими правилами:
\begin{itemize}
\item информация о линии контура хранится в виде точек

\item прямая линия контура строится по двум точкам: начальной и конечной

\item элемент дуги строится методом Безье. Начало и конец кривой описываются точками, которые лежат на самой кривой. Остальные точки, не принадлежащие кривой, служат для построения дуг
\end{itemize}

Класс \tt Font\normalfont, хранящий шрифты, созданные пользователем, предоставляющий интерфейсы загрузки и сохранения в файл, отображения в текстовом редакторе с возможностью настройки размера каждой отдельной буквы.
 
 \section* {Общая архитектура приложения}
\hspace{\parindent} Программа разработана по принципу Model-View-Controller. Глобально разделена на две подпрограммы - подпрограмму редактирования шрифта и подпрограмму тестирования созданных шрифтов.

\section* {Контроллер}
\hspace{\parindent} Реализованный класс контроллера составляет ядро бизнес-логики. Он отвечает за управление кривыми на экране исходя из передаваемых ему запросов от классов представления. Реализуемая им функциональность:
\begin{itemize}
  \item Добавление кривой Безье
  \item Перетаскивание кривой Безье по экрану
  \item Соединение кривых Безье в одну кривую
  \item Логгирование действий пользователя для возможного отката действий
  \item Очистка экрана
  \item Отмена последнего действия на холсте
\end{itemize}

\section* {Представление}
\hspace{\parindent} Считывает действия пользователя и диспетчеризует их контроллеру; предоставляет интерфейсы контроллеру для отображения визуальных данных.

\section* {Управление}
\subsection*{Редактор шрифта}
\hspace{\parindent} В подпрограмме редактирования шрифта выбор действия осуществляется переключателем Move/Draw. Режим Move позволяет двигать нарисованные кривые, Draw - рисовать новые.

Рисование кривых осуществляется нажатием левой кнопки мыши с дальнейшим движением с зажатой кнопкой указателя мыши по окну программы. Чтобы нарисовать несколько соединённых кривых, достаточно начинать каждую следующий кривую в граничной базисной точке одной из нарисованных линий. Заметим, что граничная базисная точка не должна быть уже соединена с какой-либо другой точкой, чтобы кривые зацепились.

Движение кривых можно производить либо зацепив любую базисную точку (внутренние базисные точки отображаются мелкими серыми, а граничные - крупными чёрными), либо зацепив указателем чёрную линию кривой. В первом случае будет передвинута только зацепленная точка (в случае, режима $G^1$-непрерывности будут перемещаться и соседние точки), во втором - вся кривая как единое целое. 

Соединять кривые можно, сблизив их граничные базисные точки, к которым ещё не присоединённы никакие другие кривые. Сблизить граничные точки можно передвигая кривые. 

Также имеется возможность подразбить нарисованную кривую на две части, быстро щелкнув дважды левой кнопкой мыши по чёрной линии кривой.

Отменить последнее действие можно, нажав кнопку <<Undo>>. Действия можно отменять несколько раз подряд.

Выбор гладкости кривых осуществляется флажком <<G1 smooth>>. 

После того, как редактирование буквы завершено, её можно сохранить, нажав кнопку <<Save Letter>>. Предварительно надо ввести, что за буква нарисована, в белом окне ввода <<Letter>> слева. 

В любой момент имеется возможность сохранить текущее состояние шрифта - нажав кнопку <<Save as...>> если шрифту еще не был назначен файл, и кнопку <<Save>> иначе. 

Для загрузки ранее сохранённого шрифта следует нажать кнопку <<Load>> и выбрать соответствующий файл в диалоговом окне. Поддерживается возможность редактирования уже существующих букв с последующим сохранением шрифта.

\subsection*{Редактор текста}

\hspace{\parindent} Для перехода в редактор текста, выберите вкладку <<Text Editor>> в верхней части окна справа. Вы сможете вернуться обратно в редактор шрифта, нажав на вкладку <<Font editor>>.

Для того, чтобы использовать созданный ранее шрифт, загрузите его, нажав на кнопку <<Choose font>>. После выбор шрифта, в нижней части окна появится список доступных букв. Редактор текста позволяет удобно форматировать текст с помощью нескольких видов тегов. Первый вид тегов - теги типа шрифта, определяющие шрифт текста, находящегося между ними. Второй вид тегов - теги размера шрифта, задающие размер шрифта, которым будет напечатан текст, располагающийся между этими тегами. Потяните слайдер с размером шрифта до нужного значения и нажмите <<Set size>>, выберите нужный шрифт по кнопке <<Choose font>> и печатайте текст после тегов и окне ниже. 

Набор текста осуществляется вводом с клавиатуры.

Текстовый редактор поддерживает изменение размера шрифта и самого шрифта по ходу набора текста. Так же есть возможность редактирования самих набранных символов путём редактирования текстового поля.

Ещё одним достоинством нашего редактора текста является высокая скорость  работы. Поскольку шрифт состоит из множества букв, каждая из которых состоит из множества кривых, загрузка шрифта занимает время, и при частой смене шрифта скорость отображения набранного текста значительно снижается. Поэтому мы храним уже использованные однажды в тексте шрифты в специально отведённом кэше шрифтов, не подгружая шрифт каждый раз по обращении, а только при первом использовании. 

Редактор текста позволяет сохранять и загружать для последующего редактирования набранные ранее тексты.

Ещё одно преимущество: нет необходимости вместе с файлом набранного текста хранить все используемые в нём шрифты, поскольку они автоматически импортируются и хранятся в начале файла с текстом.

\section* {Не реализовано}
\hspace{\parindent} Можно было бы сделать редактор текста, более похожий на привычный стиль Microsoft Word, добавив упрвление настройками шрифта не тегами, а элементами визуального управления. Однако, это требует значительных усилий, и не может быть выполнено сейчас в виду недостатка времени.  

Также имеет смысл добавить новые аффинные преобразования кривых: например, вращения, растяжения, и т.п.
